# Steam Game Store (Django)

Навчальний веб-проєкт на **Django**, що імітує роботу спрощеного магазину ігор у стилі **Steam**.  
Реалізовано каталог ігор, кошик без реєстрації, оформлення замовлення та адмін-панель.

---

## Функціонал

### Каталог ігор
- список доступних ігор
- назва, опис, ціна, зображення
- мінімальні та рекомендовані системні вимоги
- додавання гри в кошик

### Кошик (без реєстрації)
- працює через **Django sessions**
- збільшення кількості товарів
- видалення товарів
- автоматичний підрахунок загальної суми

### Оформлення замовлення
- введення **телефону або email**
- створення замовлення
- збереження **повного складу замовлення**
- очищення кошика після оформлення

### Адмін-панель
- керування іграми
- перегляд замовлень
- перегляд складу замовлення (Order + OrderItem)
- зміна статусу замовлення

---

## Архітектура БД

- **Product** — гра (товар)
- **CartItem** — товар у кошику
- **Order** — замовлення (контактні дані)
- **OrderItem** — склад замовлення (які ігри купили)

---

## Технології

- Python 3.14
- Django 6.0
- SQLite
- Django Templates
- HTML / CSS / JS

## Встановлення та запуск

### 1️⃣ Клонувати репозиторій
```bash
git clone https://github.com/y-lebid/steam.git
cd steam
```

### 2️⃣ Створити та активувати venv
```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
```

### 3️⃣ Встановити залежності
```bash
pip install -r requirements.txt
```

### 4️⃣ Міграції
```bash
python manage.py makemigrations
python manage.py migrate
```

### 5️⃣ Створити адміна
```bash
python manage.py createsuperuser
```

### 6️⃣ Запуск сервера
```bash
python manage.py runserver
```
